---

![text](dumpster.webp)

## PRESENTATION

dumpster is hexdump clone for (but not limited to) romeo roms.

---

## PREREQUISITES

* C11 compiler

---

## SUPPORTED PLATFORMS

- [X] Windows
- [X] Linux
- [X] macOS

---

## HOW TO USE

```sh
make example
./example > rom.hex
vim rom.hex
```
